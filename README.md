# Telegram Proxy Bot (File Storage)

## Возможности
- Flyer задания
- Кнопка «Проверить задания»
- Меню подключения прокси
- Кнопка «Поделиться»
- Рассылка только для админа
- Файловое хранилище users.json
- Экспорт users.json через /export_users

## Установка
pip install -r requirements.txt

## ENV переменные
BOT_TOKEN=xxx
FLYER_API_KEY=xxx
ADMIN_ID=123456789

## Запуск
python main.py
